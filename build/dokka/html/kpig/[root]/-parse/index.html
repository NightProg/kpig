Parse

interface Parse<T>

Interface representing a parser that processes a string input and returns a result of type T.

The Parse interface defines methods for parsing strings and handling errors. It also includes utility methods for transforming results and errors, and for obtaining parsing results along with their locations.

The type of the result produced by the parser.
